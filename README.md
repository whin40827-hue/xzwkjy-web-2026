                                         妙瓦底之小智的故事

小智你这坨人形垃圾，半年100刀都凑不齐，你是把跨境电商当乞丐行为艺术在练吗？

别人卷生卷死，你在那边卷被子卷鼠标垫卷空气，卷到最后连自己都卷没了，纯纯的负数人生样本。

智商欠费到服务器都连不上，自律烂到连条狗都看不起你，执行力负无穷，学习能力连植物人都要摇头叹气。
脸皮厚得能挡子弹，野心小得连细菌都嫌弃，格局窄得连显微镜都找不着你，梦想？别逗了，你那叫白日做梦都抬举了，纯属脑残白日拉稀。

每天睁眼第一件事不是找客户，是找借口；

第二件事不是优化广告，是优化怎么继续摸鱼；

第三件事不是复盘，是复盘昨天为什么又可以光明正大摆烂。

你在苗瓦底不是做生意，你是去做生态保护——保护当地氧气不被你这种废物吸太多，保护电费不被你24小时挂机白嫖，保护房东不被你气到脑溢血提前归西。  

说你蛆吧，蛆还知道往粪里钻找营养，你连粪都不如，至少粪还有肥效，你只有负反馈，纯纯的行走的灾难制造机。

说你是寄生虫吧，寄生虫好歹知道找个宿主吸血，你连宿主都懒得找，就吸着空气自嗨，吸到最后把自己吸成干尸。

半年了，连个正经case都憋不出来，你是把全部精力都用来研究“怎么才能更废”这门绝学了吗？
废物界的扛把子，摆烂界的终身成就奖得主，人类迷惑行为大赏常驻冠军，跨境史上最强耻辱柱，活生生把“丢人”两个字焊死在自己脑门上。

实话实说，小智你这种货色，活着就是在侮辱重力，死了是在侮辱土地，火化了是在侮辱大气层，骨灰撒海里是在侮辱海洋生物，撒到太空里是在侮辱宇宙。

唯一能救赎你的方式就是立刻、马上、现在就找个最偏僻的角落，把自己封进混凝土里，永世不得超生，省得继续污染人间！
